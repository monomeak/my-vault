>[!report]
>Allure Report is used to present automated test result in a clear and structure way. Playwright executes the test cases, then Allure collects the result data such as passed cases, failed cases, screenshots, traces, error messages, and execution time.

In Jenkins, the Allure Plugin automatically publishes the report after the pipeline is finishes. This helps the team quickly review the testing result, identify failed scenarios, debugs issues, and understand the quality status of the application before release.
