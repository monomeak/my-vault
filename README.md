# Obsidian Vault

My personal Obsidian vault for storing notes, documentation, learning resources, and project ideas.

## Open the Vault

1. Clone this repository:

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

2. Open Obsidian.
3. Select **Open folder as vault**.
4. Choose the cloned folder.

## Sync

This vault uses the **Obsidian Git** plugin to automatically commit and push changes to GitHub.

## Repository Structure
